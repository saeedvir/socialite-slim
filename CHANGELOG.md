# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-13

- Telegram driver support
- Configuration file for easy setup
- Example controller and routes
- Usage documentation

- Removed unused OAuth providers (Facebook, Twitter, LinkedIn, etc.)
- Simplified composer.json dependencies
- Updated README.md with new package information

### Removed
- OAuth 1.0 support (Twitter provider)
- Unused provider implementations
- Unnecessary dependencies
